# Todo App

## Project Setup

### Prerequisites
- Android Studio
- JDK 11 or higher
- Kotlin plugin

### Running Tests
To run tests, use the following Gradle command:
```bash
./gradlew test
```

### Features
- Add, edit, and delete todo items
- Persistent local storage
- User feedback for actions

### Architecture
- MVVM Architecture
- Room Database
- Kotlin Coroutines
- Unit Tested
